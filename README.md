# web-actividad-1

Despues de haber realizado la aplicacion de la actividad 1, pude al final utilizar muchos de los elementos que aprendi en clase, para asi poder realizar un mini aplicacion con una interfaz moderna, clara y funcional, acerca del manejo de habitos que puede registrar una persona y asi darle seguimiento. Para poder hacer esto, primero decidi en usar algunos de los componentes reutilizables que React me podria ofrecer, esto debido para poder realizar una estructura organizada y facil de mantener dentro de la aplicacion. Algunos de estos componentes fueron: PrimaryButton, InfoCard y SidebarMenu. Decidi basicamente usar estos componentes, debido a que estos pudieron facilitar mucho el proceso de crear el fontend de manera facil y segura, con PrimaryButton siendo la estructura de un boton, InfoCard siendo la estructura de una carta con informacion y SidebarMenu siendo un menu deslizante con informacion. Eso sin mencionar de que estos componentes tambien se pueden reutilizar dentro del codigo mas de una vez, para asi no repetir codigo en la parte principal del codigo. Luego en terminos de dise単o y estructura, decidi basicamente realizar un estructura de layout compuesto, en donde el Sidebar y el Contenido estuvieran divididos en un solo row, para asi poder facilitar mucho la navegacion de la aplicacion sin usar tanto espacio en el frontend de la aplicacion, ademas decidi usar Tailwind CSS para el dise単o visual de la aplicacion, ya que este ofrecia muchos commandos de utilidad para asi poder especificar facilmente como seria los colores, tama単os, alineamiento y posiciones en ciertas partes del fontend de la aplicacion, asi como breakpoint como md y sm lo cual hacia que la aplicacion pudiera ser adaptado en cualquier tipo de plataforma. Ya por ultimo, para asi darle un poco mas de personalidad en mi pagina decidi basicamente crear una pagina de manejo de habitos, en donde las personas pueden crear habitos personalizados y de esta manera realizarlos para asi manejar mejor sus actividades y asi completar estos habitos a diario. Yo decidi hacer este tipo de aplicacion, ya que este se me hizo una aplicacion muy facil de hacer, debido a la estructura que realize despues de haber hecho el dise単o, ademas decidi usar estados interactivos tales como hover y focus-visible para asi poder mejorar el aspecto estetico de la aplicacion al momento de navegar por areas, ya se en teclado o en mouse.
